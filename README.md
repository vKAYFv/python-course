# 🐍 Python с нуля — Практический курс

> **Полный курс Python на русском языке** — от первой строчки кода до реальных проектов.  
> Работает прямо в браузере через GitHub Codespaces — ничего устанавливать не нужно.

---

## 🚀 Быстрый старт

### Вариант 1: GitHub Codespaces (рекомендуется)
1. Нажми зелёную кнопку **`<> Code`** → вкладка **Codespaces**
2. Нажми **"Create codespace on main"**
3. Подожди ~1 минуту — среда поднимется автоматически
4. Открой любой модуль и начинай!

### Вариант 2: Локально
```bash
git clone https://github.com/vKAYFv/python-course.git
cd python-course
python --version  # нужен Python 3.10+
```

## 🗂 Структура каждого модуля

```
modules/XX_название/
├── README.md        ← Теория с объяснениями и примерами
├── theory.py        ← Код из теории (можно запускать)
├── tasks.py         ← Задания (TODO-заглушки для тебя)
├── tasks_test.py    ← Автотесты — проверь себя!
└── solution.py      ← Решения (загляни только после попытки!)
```

---

## ✅ Как работать с курсом

1. **Читай теорию** — открой `README.md` модуля
2. **Запускай примеры** — `python theory.py`
3. **Решай задачи** — открой `tasks.py`, заполни функции
4. **Проверяй себя** — `python -m pytest tasks_test.py -v`
5. **Застрял?** — загляни в `solution.py`

---

## 🧪 Запуск тестов

```bash
# Один модуль
cd modules/01_basics
python -m pytest tasks_test.py -v

# Весь курс
python -m pytest modules/ -v
```

---

## 💡 Советы

- Не подглядывай в `solution.py` сразу — попробуй сам хотя бы 15 минут
- Если тест не проходит — читай текст ошибки внимательно, Python говорит прямо
- Экспериментируй — меняй код, ломай, чини
