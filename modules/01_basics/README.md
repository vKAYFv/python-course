## 1.1 Твоя первая программа

Программа на Python — это просто текстовый файл с инструкциями. Python читает их сверху вниз и выполняет.

```python
print("Привет, мир!")
```

`print()` — встроенная функция. Она выводит текст в терминал.

## 1.2 Переменные

Переменная — это **имя**, за которым хранится значение. Думай о ней как о ящике с наклейкой.

```python
имя = "Алексей"
возраст = 25
рост = 1.82

print(имя)      # Алексей
print(возраст)  # 25
```

> ⚠️ Python чувствителен к регистру: `имя` и `Имя` — разные переменные.

**Правила именования:**
- Только буквы, цифры, `_`
- Не начинается с цифры
- Нет пробелов (используй `_` вместо пробела)
- Нельзя использовать ключевые слова (`if`, `for`, `while` и др.)

```python
# ✅ Хорошо
user_name = "Катя"
total_price = 1500
is_active = True

# ❌ Плохо
2user = "Петя"      # начинается с цифры
my name = "Ваня"    # пробел
```

## 1.3 Типы данных

Python автоматически определяет тип данных:

```python
# int — целые числа
количество = 10
год = 2024

# float — числа с точкой
цена = 99.99
температура = -5.5

# str — строки (текст)
имя = "Мария"
сообщение = 'Привет!'  # одинарные или двойные кавычки

# bool — булевы значения
активен = True
удалён = False
```

Узнать тип переменной:
```python
x = 42
print(type(x))  # <class 'int'>

y = "hello"
print(type(y))  # <class 'str'>
```

## 1.4 Арифметические операторы

```python
a = 10
b = 3

print(a + b)   # 13  — сложение
print(a - b)   # 7   — вычитание
print(a * b)   # 30  — умножение
print(a / b)   # 3.333... — деление (всегда float)
print(a // b)  # 3   — целочисленное деление
print(a % b)   # 1   — остаток от деления
print(a ** b)  # 1000 — возведение в степень
```

---

## 1.5 Операторы сравнения

Возвращают `True` или `False`:

```python
x = 10
y = 20

print(x == y)   # False — равно?
print(x != y)   # True  — не равно?
print(x < y)    # True  — меньше?
print(x > y)    # False — больше?
print(x <= 10)  # True  — меньше или равно?
print(x >= 10)  # True  — больше или равно?
```

## 1.6 Ввод данных

`input()` читает ввод пользователя. Всегда возвращает **строку**!

```python
имя = input("Как тебя зовут? ")
print("Привет,", имя)
```

Если нужно число — преобразуй:
```python
возраст = int(input("Сколько тебе лет? "))
рост = float(input("Твой рост в метрах? "))

print(f"Через 10 лет тебе будет {возраст + 10} лет")
```

## 1.7 f-строки (форматирование)

Самый удобный способ вставить переменную в строку:

```python
имя = "Дима"
возраст = 22

# f-строка — ставь f перед кавычкой
print(f"Меня зовут {имя}, мне {возраст} года")

# Можно делать вычисления прямо внутри
print(f"Через год мне будет {возраст + 1}")
print(f"Имя из 4 букв: {'Да' if len(имя) == 4 else 'Нет'}")
```

## 1.8 Условия (if / elif / else)

```python
оценка = 75

if оценка >= 90:
    print("Отлично!")
elif оценка >= 70:
    print("Хорошо")
elif оценка >= 50:
    print("Удовлетворительно")
else:
    print("Нужно повторить материал")
```

> ⚠️ **Отступы важны!** Python использует 4 пробела (или Tab) для обозначения блоков кода.

## 1.9 Циклы

### while — повторяй пока условие True

```python
счётчик = 1

while счётчик <= 5:
    print(f"Итерация {счётчик}")
    счётчик += 1  # то же что счётчик = счётчик + 1
```

### for — перебор по последовательности

```python
# range(5) даёт числа 0, 1, 2, 3, 4
for i in range(5):
    print(i)

# range(1, 6) даёт числа 1, 2, 3, 4, 5
for i in range(1, 6):
    print(f"Число: {i}")

# range(0, 10, 2) — с шагом 2
for i in range(0, 10, 2):
    print(i)  # 0, 2, 4, 6, 8
```
