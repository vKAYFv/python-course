## 2.1 Что такое строка

Строка — это последовательность символов. В Python строки **неизменяемы**: раз созданная строка не меняется, все операции создают новую.

```python
s = "Привет, Python!"
print(len(s))       # 15 — длина строки
print(s[0])         # П — первый символ
print(s[-1])        # ! — последний символ
```

## 2.2 Срезы (slicing)

```python
s = "Python"
#    012345   ← индексы
#   -6-5-4-3-2-1 ← отрицательные индексы

print(s[0:3])    # Pyt  — символы с 0 по 2
print(s[2:])     # thon — с 2-го до конца
print(s[:4])     # Pyth — с начала до 3-го
print(s[-3:])    # hon  — последние 3 символа
print(s[::2])    # Pto  — каждый второй
print(s[::-1])   # nohtyP — строка наоборот!
```

## 2.3 Основные методы строк

```python
s = "  Привет, Мир!  "

# Регистр
print(s.upper())        # ПРИВЕТ, МИР!
print(s.lower())        # привет, мир!
print(s.title())        # Привет, Мир!
print(s.capitalize())   # Привет, мир!

# Пробелы
print(s.strip())        # "Привет, Мир!"   — обрезает с обеих сторон
print(s.lstrip())       # "Привет, Мир!  " — только слева
print(s.rstrip())       # "  Привет, Мир!" — только справа

# Поиск и замена
print(s.find("Мир"))       # 9  — индекс первого вхождения, -1 если нет
print(s.count("и"))        # 2  — количество вхождений
print(s.replace("Мир", "Python"))  # заменяет все вхождения

# Проверки
print("hello".startswith("he"))   # True
print("hello".endswith("lo"))     # True
print("abc123".isalnum())          # True — только буквы и цифры
print("   ".isspace())             # True — только пробелы
print("123".isdigit())             # True — только цифры
```

## 2.4 Разбивка и склейка

```python
# split — разбивает строку в список
csv = "яблоко,груша,апельсин"
фрукты = csv.split(",")
print(фрукты)  # ['яблоко', 'груша', 'апельсин']

текст = "раз два три"
слова = текст.split()   # по пробелам
print(слова)  # ['раз', 'два', 'три']

# join — склеивает список в строку
print(" | ".join(фрукты))  # яблоко | груша | апельсин
print("".join(["a", "b", "c"]))  # abc
```

## 2.5 Форматирование строк

### f-строки (рекомендуется)
```python
name = "Алиса"
score = 95.5

print(f"Игрок: {name}")
print(f"Счёт: {score:.1f}")       # 1 знак после точки → 95.5
print(f"Счёт: {score:08.2f}")     # 8 символов, заполнить нулями → 00095.50
print(f"{'Заголовок':^30}")       # центрировать в 30 символах
print(f"{'Левый':<20}|{'Правый':>20}")  # выравнивание
```

### Метод .format()
```python
print("Привет, {}! Тебе {} лет.".format("Боря", 25))
print("{name} набрал {score} очков".format(name="Катя", score=100))
```

## 2.6 Экранирование символов

```python
print("Он сказал: \"Привет!\"")   # кавычки внутри
print('Это апостроф: it\'s')
print("Первая строка\nВторая строка")   # \n — перенос
print("Колонка1\tКолонка2")              # \t — табуляция
print("C:\\Users\\username")            # \\ — обратный слэш
print(r"C:\Users\raw")                  # r-строка — без экранирования
```

## 2.7 Строки — это последовательности

```python
s = "Python"

# in — проверка вхождения
print("Py" in s)      # True
print("py" in s)      # False — регистр важен!

# Перебор символов
for символ in "ABC":
    print(символ)

# enumerate — индекс + символ
for i, символ in enumerate("Hi"):
    print(f"{i}: {символ}")
```

## 📌 Шпаргалка методов

| Метод | Что делает |
|-------|-----------|
| `s.upper()` | Всё в верхний регистр |
| `s.lower()` | Всё в нижний регистр |
| `s.strip()` | Убрать пробелы по краям |
| `s.split(sep)` | Разбить в список |
| `sep.join(list)` | Склеить список |
| `s.replace(a, b)` | Заменить a на b |
| `s.find(sub)` | Найти подстроку |
| `s.startswith(p)` | Начинается с p? |
| `s.count(sub)` | Сколько раз встречается |
| `len(s)` | Длина строки |
